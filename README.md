This depository contains the data and R script used in the paper

Wagner, V., Botta-Dukat, Z., Sabatini, F.M., Pillar, V.D. (2026) "Applied Vegetation Science as a publication venue for diverse research themes" Applied Vegetation Science.

(1) AVS_output_analysis.R:  R script

(2) AVS_publications_1_1452_analysis.txt: 
File with information on papers published in Applied Vegetation Science. 
Publication records were extracted from Clarivate Web of Science on December 18, 2025, filtering for "Applied Vegetation Science" as a journal.
